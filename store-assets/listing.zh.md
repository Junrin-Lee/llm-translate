# 商店文案 — 简体中文

**名称:** LLM Translate

**类别:** 效率 (Productivity)

**简介**(≤132 字符):

> 用你自己的 OpenAI 兼容或 Anthropic 兼容 API Key 做划词翻译与全文翻译。无账号、无后端、无遥测。

## 详细描述

LLM Translate 把大模型翻译带到任意网页——用的是**你自己的** API Key。接入 OpenAI 兼容或
Anthropic 兼容端点(官方 API、网关,或你自建的代理),在阅读处即时翻译。

**划词翻译**
选中任意文本,即时流式翻译、就地展示。单词或短语显示词典卡片(音标、词性、多义、例句);
句子显示纯译文。可复制、重试,或随时切换目标语言。

**全文翻译**
整页翻译,支持双语对照(译文插在每段下方,保留原文)与仅译文模式(可一键还原)。随滚动
懒加载、跟随单页应用(SPA)导航,并提供页内工具条(进度、取消、还原)。

**你的密钥,你的端点,你的隐私**
- 自带 OpenAI 兼容或 Anthropic 兼容 API Key。
- 所有配置只保存在你本机——无账号、无需登录。
- 唯一的网络请求发往你配置的端点;开发者不运营任何服务器、不收集任何数据。无遥测。

**贴合你的使用习惯**
- 多条服务商配置,支持全局默认与按功能覆盖。
- 触发方式:浮动图标、选中即翻、或快捷键。
- 右键菜单,以及"始终翻译"站点清单。
- 可自定义 Prompt 模板、站点禁用清单,JSON 导入导出(默认不含 API Key)。
- 界面支持中文与英文。
