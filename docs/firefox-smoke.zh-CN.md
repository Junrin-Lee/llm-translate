# Firefox 发布冒烟清单

[English](./firefox-smoke.md) · **简体中文**

每次提交 AMO 前,在最新正式版 Firefox 上执行本清单——通过 about:debugging 临时装载 `pnpm zip:firefox` 产出的 zip(或已签名的构建产物)。

## 安装与权限
- [ ] 全新安装且站点访问已授权:不会打开 onboarding 页,工具栏无角标。
- [ ] 全新安装且拒绝站点访问:onboarding 页自动打开;点击授权按钮可正常工作;页面切换到成功态。
- [ ] 在 about:addons 中撤销站点访问:工具栏图标出现红色 "!" 角标;popup 与设置页均显示警示条。
- [ ] 从 popup 警示条内点击授权:无需重新打开 popup,警示条即消失;角标同步清除。
- [ ] 未授权状态下,在任意页面使用快捷键 / 右键菜单:自动打开 onboarding 页(若已打开则聚焦该标签页)。

## 核心功能(对照 Chromium 端 e2e)
- [ ] 划词翻译:选中即翻流式输出译文卡片;选中单词/短语渲染词典卡片。
- [ ] 通过工具栏 popup 触发全文翻译:双语对照模式在原文块下方插入译文;仅译文模式原地替换且可还原。
- [ ] 右键菜单可翻译整页 / 所选内容;翻译后菜单标题切换为 "还原原文"。
- [ ] 快捷键 Ctrl+Shift+S / Ctrl+Shift+P 均可用(可在 about:addons → 齿轮图标 → 管理扩展快捷键 中重新绑定)。
- [ ] 设置页:服务商增删改查、对真实端点测试连接、导入导出可往返、缓存面板可清空两种缓存。

## Firefox 专属
- [ ] `storage.session` 划词缓存在会话内持续有效,完整重启浏览器后即清空。
- [ ] 在 https 页面上流式输出正常(background 通过 host permission 发起请求,无 CORS 失败)。
- [ ] popup 渲染正常(与 Chrome 相比无滚动条/裁切差异)。
